# 🏥 Healthcare Management System (SQL Project)

## 📌 Description
This project is an advanced healthcare database system using SQL. It manages patients, doctors, appointments, billing, insurance, and hospital operations.

## 🧱 Tables Used
- Patients
- Doctors
- Departments
- Appointments
- Treatments
- Bills
- Payments
- Rooms
- Admissions
- Insurance
- Insurance Claims

## 🔥 Features
- Appointment scheduling
- Billing and payment tracking
- Insurance claim management
- Room allocation system
- Doctor performance tracking

## ⚙️ SQL Concepts Used
- Joins (INNER, LEFT)
- Subqueries
- Window Functions (RANK)
- Triggers
- Views
- GROUP BY & HAVING

## 🚀 How to Run
1. Open SQL environment (MySQL/SQLite)
2. Copy and run the SQL file
3. Execute queries
